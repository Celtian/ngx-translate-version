## Install

1. Use yarn (or npm) to install the package

```terminal
yarn add ngx-translate-version
```

2. Add NgxRepeatModule into your module `imports`

```typescript
  import { NgxRepeatModule } from 'ngx-translate-version';

  @NgModule({
   // ...
   imports: [
     // ...
     NgxRepeatModule
   ]
  })
```
